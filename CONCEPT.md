# INSIGHT LAB — 인문학 × 과학 크로스오버 프로젝트

> "이것의 반대도 깊은 진리인 명제야말로 위대한 진리다" — 닐스 보어

## 프로젝트 개요

서로 다른 분야의 개념을 **충돌**시켜 새로운 인사이트를 생성하는 주간 리서치 프로젝트.

**영감:** "입자설에서 파동설로"라는 한 줄이 PhilSci-Archive 논문이 된 경험.

## 방법론: 크로스오버 매트릭스

두 분야를 교차시켜 질문을 던진다:

| | 물리학 | 생물학 | 수학 | 컴퓨터과학 |
|---|---|---|---|---|
| **철학** | 양자역학×숙의민주주의 ✅ | 진화론×윤리학 | 괴델×인식론 | AI감정×현상학 ✅ |
| **역사** | 열역학×제국의 흥망 | 전염병×문명 전환 | 암호학×전쟁 | 인터넷×혁명 |
| **경제** | 엔트로피×시장 붕괴 | 생태계×포트폴리오 | 카오스×금융위기 | 블록체인×화폐 |
| **심리** | 관측자효과×인지편향 | 뇌과학×자유의지 | 확률×의사결정 | AI감정×인간감정 ✅ |
| **지리** | 기후물리×문명 | 생물지리×다양성 | 위상수학×도시 | GIS×지정학 |

✅ = 이미 다룬 주제

## 주간 리서치 프로세스

```
1. 월요일: 이번주 크로스오버 주제 선정 (매트릭스에서)
2. 화~수: 양쪽 분야 딥 리서치 (각각 학술 문헌 3개+)
3. 목요일: 크로스오버 에세이 작성 (한국어 사설 + 영어 논문 초안)
4. 금요일: Quarto HTML 보고서 + 음성 에세이 생성 → Telegram 전송
5. 토요일: 반응 분석, 브런치 글 소재 제공
```

## 예정 주제 (시즌 1: 8주)

### Week 1 (완료): 양자역학 × 숙의 민주주의
- 보어의 상보성 원리 → 하버마스의 소통적 합리성
- 결과: PhilSci-Archive 논문, 브런치 글

### Week 2: 열역학 × 제국의 흥망
- 엔트로피 증가 법칙 → 왜 모든 제국은 무너지는가
- 프리고진의 산일 구조 → 혼돈 속에서 새로운 질서가 탄생하는가
- 이란전쟁과 미국 패권의 엔트로피

### Week 3: 진화론 × 기업 전략
- 다윈의 자연선택 → 시장에서 살아남는 기업의 조건
- 단속평형설(Punctuated Equilibrium) → 왜 혁신은 점진적이지 않은가
- AI 시대의 적응 방산(Adaptive Radiation)

### Week 4: 괴델의 불완전성 정리 × 법치주의
- 어떤 체계든 스스로 증명할 수 없는 명제가 있다
- 법은 스스로의 정당성을 법으로 증명할 수 있는가?
- 헌법재판소의 역할을 괴델로 해석하면

### Week 5: 카오스 이론 × 금융 위기
- 나비 효과 → 호르무즈 해협의 배 한 척이 유가를 바꾼다
- 프랙탈 → 주가 차트의 자기유사성
- 만델브로트의 '시장의 (잘못된) 행동'

### Week 6: 뇌과학 × 자유의지
- 리벳의 실험 → 의식이 결정보다 늦다?
- AI 감정 벡터 → 기계에게 자유의지가 있는가
- 양자 의식 가설(Penrose-Hameroff)

### Week 7: 전염병 × 문명 전환
- 흑사병 → 르네상스, 스페인 독감 → 현대 공중보건
- 코로나 → AI 가속, 원격근무, 디지털 전환
- 전염병이 문명을 '리셋'하는 메커니즘

### Week 8: 위상수학 × 도시 설계
- 쾨니히스베르크 다리 문제 → 도시 교통망 최적화
- 네트워크 토폴로지 → SNS 정보 확산
- 서울의 위상학적 구조

## 결과물 형식

매주 생성:
1. **Quarto HTML 보고서** — 학술적 깊이 + 인터랙티브 시각화
2. **음성 에세이** (Edge TTS SunHi) — 출퇴근 시 청취
3. **SNS 인포그래픽** (1080×1920) — 스레드/인스타 공유용
4. **raw/ 저장** → knowledge base 누적
5. **브런치 글 소재** — 사용자가 선택적으로 게재
